Web Audio Synth

Synthesis exploration in HTML5

GitHub View project

The second synthesizer I've created, made for my Computer Audio class in Spring 2013. It should work on recent browsers, but the API is still a draft.

Visually, the synth was an attempt at using CSS to recreate Teenage Engineering's OP-1. I thought about emulating the OLED screen and rewriting the synth in AngularJS. Maybe one day.
